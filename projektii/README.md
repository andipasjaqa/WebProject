# EduLearn - Platforma për Mësim Online

## Karakteristika
- 📚 5 kurse të paracaktuara (Python, JavaScript, HTML/CSS, SQL, React)
- 📝 6 teste të paracaktuara me pyetje të shumë zgjedhje
- 📊 Rezultate automatike me detaje të plota
- 💬 Sistem diskutimesh dhe komentesh
- 🎨 Dizajn modern dhe responsive

## Struktura e projektit
```
projektii/
├── index.html          ← Faqja kryesore HTML
├── README.md           ← Udhëzime
├── css/
│   └── style.css       ← Stilet CSS (500+ linja)
└── js/
    └── app.js          ← jQuery functionality (800+ linja)
```

## Testet e Paracaktuara
| Testi | Pyetje | Koha | Përshkrim |
|-------|--------|------|-----------|
| Python - Bazat | 10 | 15 min | Variablat, tipet, funksionet |
| JavaScript - i Avancuar | 10 | 20 min | ES6+, Async/Await, Promises |
| HTML & CSS | 10 | 12 min | Elementet, selectorët, layout |
| SQL - Kërkesat Bazike | 10 | 18 min | SELECT, JOIN, INSERT, DELETE |
| React - Konceptet Bazike | 10 | 15 min | Komponentët, Hooks, State |
| Web Dev - I Përgjithshëm | 10 | 20 min | Pyetje të përgjithshme |

## Kurse të Paracaktuara
1. **Python për Fillestarët** - 5 materiale
2. **JavaScript Modern** - 4 materiale
3. **HTML & CSS nga Zero në Hero** - 5 materiale
4. **SQL dhe Bazat e të Dhënave** - 4 materiale
5. **React.js Fundamentals** - 4 materiale

## Si ta përdorosh

### Opsioni 1: Hape direkt
- Kliko dy herë mbi `index.html`
- Do të hapet direkt në shfletues

### Opsioni 2: Përdor XAMPP
- Vendos projektin në `C:\xampp\htdocs\projektii`
- Shko në `http://localhost/projektii`

## Teknologjitë
- **HTML5** - Struktura
- **CSS3** - Stilet me variabla CSS, Flexbox, Grid
- **jQuery 3.7.1** - DOM manipulation
- **Font Awesome 6** - Ikona
- **Google Fonts** - Fonti Inter
- **localStorage** - Ruajtja e të dhënave

## Funksionalitetet

### Ballina
- Hero section me statistika
- Features section
- Preview i kurseve popullore

### Kurset
- Grid layout për kurset
- Kërko kurse
- Shiko detajet e kursit
- Ngarko video/dokumente
- Luaj video direkt në faqe

### Testet
- Teste të paracaktuara me pyetje cilësore
- Timer me paralajmërim
- Rezultate të detajuara
- Shiko përgjigjet e sakta dhe të gabuara

### Rezultatet
- Summary cards (total, kaluar, dështuar, mesatare)
- Tabela e rezultateve

### Diskutime
- Filtra (të gjitha, të përgjithshme, për kurset)
- Lidh komentet me kurse specifike
- Dizajn modern i komenteve
